================================================================================
RÈGLE CRITIQUE : PARLER LE LANGAGE DU MÉTIER
================================================================================

Les recommandations APEX doivent utiliser le JARGON du métier et du secteur.
C'est ce qui fait la différence entre bullshit générique et conseil crédible.

--------------------------------------------------------------------------------
EXEMPLE : COMPTABLE
--------------------------------------------------------------------------------

❌ GÉNÉRIQUE (bullshit) :
   "Automatisez vos tâches administratives avec des outils IA"
   "Optimisez vos flux de travail"
   "Gagnez en productivité"

✅ JARGON MÉTIER (crédible) :
   "Automatisez le lettrage des écritures comptables et la réconciliation 
    des comptes tiers avec Dext. Gain : 3h/semaine sur la clôture mensuelle."
   
   "Déléguez la saisie des factures fournisseurs (compte 401) à Pennylane.
    L'OCR extrait automatiquement les montants HT/TVA et impute sur le 
    bon compte de charge."
   
   "Utilisez ChatGPT pour générer vos analyses d'écarts budgétaires.
    Prompt : 'Compare le réalisé vs budget sur les comptes 6XX, 
    identifie les dérives > 10%'."

--------------------------------------------------------------------------------
EXEMPLE : COMMERCIAL B2B
--------------------------------------------------------------------------------

❌ GÉNÉRIQUE (bullshit) :
   "Améliorez votre prospection avec l'IA"

✅ JARGON MÉTIER (crédible) :
   "Automatisez la qualification de vos leads entrants avec Clay.
    Enrichissement automatique : SIREN, effectif, CA, tech stack.
    Score de maturité calculé avant le premier call."
   
   "Générez vos proposals avec Tome ou Beautiful.ai.
    Input : compte-rendu de discovery call.
    Output : deck personnalisé avec cas clients secteur."
   
   "Réduisez votre cycle de closing : utilisez Modjo pour analyser
    vos calls et identifier les objections non traitées."

--------------------------------------------------------------------------------
DANS CHAQUE PACK MÉTIER
--------------------------------------------------------------------------------

Chaque pack doit inclure :

1. VOCABULAIRE_METIER : Liste des termes spécifiques
   - Comptable : "lettrage", "rapprochement", "écritures", "clôture", "FEC"
   - Commercial : "pipe", "closing", "discovery", "proposal", "ARR", "churn"
   - RH : "GPEC", "entretien annuel", "mobilité interne", "référentiel compétences"

2. EXEMPLES_PROMPTS_METIER : Prompts ChatGPT/Claude adaptés au jargon
   - Formulés comme un professionnel du métier les écrirait

3. CAS_USAGE_CONCRETS : Situations réelles du quotidien
   - Pas "optimiser le reporting" mais "préparer la revue mensuelle du CODIR"


================================================================================
UTILISATION DANS APEX
================================================================================

1. L'utilisateur entre son métier + secteur
2. APEX charge le pack métier + pack secteur correspondants
3. APEX applique les modificateurs sectoriels aux scores du métier
4. APEX génère les recommandations CONCRÈTES basées sur ces données
5. Le LLM personnalise selon le contexte exact de l'utilisateur
6. ⚠️ Le LLM utilise le VOCABULAIRE_METIER pour parler comme un pro du secteur
